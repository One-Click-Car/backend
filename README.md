# One-Click-Car AI - Backend

This is a NextJS application for One-Click-Car AI - an AI-powered car marketplace.

## Features

- AI-Powered Natural Language Search
- Dynamic Car Listing Display
- Comprehensive Car Detail View
- "Find My Car" / "Sell My Car" Request Form
- Recent Searches & Saved Matches

## Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:9002](http://localhost:9002) to view the app.

## Tech Stack

- Next.js 15 with Turbopack
- Firebase Realtime Database
- Genkit + Google GenAI
- Tailwind CSS + Radix UI
